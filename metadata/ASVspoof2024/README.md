# ASVspoof2024

## Overview
**ASVspoof2024** (ASVspoof 5) is the fifth edition in a series of challenges that promote the study of speech spoofing and deepfake attacks. The dataset is built from crowdsourced data collected from approximately **2,000 speakers** in diverse acoustic conditions, making it suitable for developing and evaluating systems that detect synthetic and manipulated speech.

## Supported Tasks
1. **Synthetic / Fake Speech Detection**

---

---

## Data Format

Each sample is stored as a JSON entry with the following fields:

| Field | Description |
|------|-------------|
| `id` | Unique utterance ID |
| `path` | Path to audio file |
| `sampling_rate` | Audio sampling rate |
| `duration` | Audio duration in seconds |
| `dataset` | Source dataset |
| `attack_type` | Type of spoofing attack (null for bonafide) |
| `real_or_fake` | Ground-truth label for authenticity |

### Label Definitions

## Notes
- All audio files are sampled at **16 kHz**.
- Audio files are stored in **FLAC format** for lossless compression.
- This is a **binary classification** task where each recording is either bonafide (genuine) or spoof (fake).
- The `attack_type` field identifies the specific spoofing method used (e.g., A01, A02, A05) and is `null` for bonafide samples.
- The dataset includes diverse acoustic conditions and speaker demographics, making it suitable for evaluating generalization capabilities of deepfake detection systems.
- Duration varies across samples, ranging from a few seconds to over 20 seconds.
